# Assets

This directory contains graphics, audio, and other media files for the mod.

## Structure

- `flags/` - Flag designs and coat of arms
- `gfx/` - Graphics files (UI elements, portraits, etc.)
- `music/` - Audio files

## File Formats

- Images: .dds, .png
- Audio: .ogg, .mp3
- Flags: Scripted coat of arms system (see docs)

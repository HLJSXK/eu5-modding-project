# EU5 模组项目简明指南

**版本:** 1.0
**更新日期:** 2026年1月21日

## 1. 项目简介

欢迎来到我们的《欧陆风云5》（EU5）私人模组项目！

这个项目是为我们朋友小团体量身打造的，主要目标是让大家能更方便、更有趣地一起玩EU5。无论你是不是电脑高手，这个项目都力求做到**简单易用、一键操作**。我们希望通过这个项目，解决联机和游戏内容的一些痛点，让我们的联机体验更加丝滑，游戏内容更加丰富多彩。

## 2. 主启动器介绍

为了方便大家使用，我们开发了一个专属的图形界面启动器。你不需要去研究复杂的文件替换或者设置，**所有核心功能都将集成在这个启动器里**。

你只需要打开这个主程序，就可以看到清晰的选项，用于开启或关闭我们需要的各项功能。后续所有新增的功能，也都会集成到这里。

## 3. 核心功能一：便捷的局域网联机

### 3.1 功能说明

这个功能主要是为了解决我们远程联机时，因为网络问题（比如NAT类型）导致的无法加入游戏、延迟高等问题。通过本功能，我们可以模拟一个“虚拟局域网”，让天各一方的大家感觉就像在同一个网吧里玩一样流畅。

### 3.2 实现原理（可跳过）

为了实现这个功能，启动器会自动帮你完成以下三步操作：

1.  **替换游戏文件**：启动器会将EU5游戏目录下的 `steam_api.dll` 文件，临时替换为一个名为 “Goldberg Steam Emulator” 的特殊版本。这个文件能让游戏在不通过Steam官方服务器的情况下，也能识别到局域网内的其他玩家。
2.  **部署n2n客户端**：启动器会直接集成开源项目 [n2n](https://github.com/ntop/n2n) 的客户端功能。它会在你的电脑上创建一个虚拟网卡，并自动连接到我们预先部署好的`supernode`中心服务器，从而形成一个加密的虚拟网络。
3.  **自动配置**：你完全不需要关心`supernode`服务器的地址、端口或连接密码，这些信息都已经预置在启动器中，实现了全自动配置。

当你退出局域网模式时，启动器会自动将所有文件和设置恢复原状，不影响你正常游玩单机或加入其他非本项目的联机游戏。

### 3.3 如何使用

在主启动器界面上，你会看到一个类似 **“开启/关闭局域网模式”** 的按钮。点击“开启”，等待程序提示成功后，直接启动游戏即可在局域网列表里看到朋友创建的房间。游戏结束后，点击“关闭”即可恢复。

## 4. 核心功能二：我们的私人定制Mod

### 4.1 功能说明

玩得久了，大家可能会觉得原版游戏里不同国家玩起来差别不大，或者觉得某些热门Mod的某个功能很棒，但又不想装一大堆Mod导致冲突和卡顿。我们的私人Mod就是为了解决这个问题。

### 4.2 Mod目标

1.  **功能整合**：我们会挑选一些广受好评的大型Mod（例如“权游”、“天堂之令”等）的核心功能，经过兼容性处理后，整合到我们的私人Mod中。
2.  **增加风味**：我们会为我们常玩的国家（例如大明、法兰西、奥斯曼等）添加一系列专属的决议、事件链和任务，让每个国家的历史进程和游戏体验更加独特，告别“千篇一律”。

### 4.3 如何使用

这个功能同样由主启动器管理。在启动器上，你可以选择**加载/卸载私人Mod**。启动器会自动将Mod文件安装到正确的位置，并在游戏启动项中为你勾选好。你只需要点一下按钮，就可以带着我们专属的内容进入游戏。

# Computes per-market scarcity flags for all pop-demand substitute groups.
#
# Unified rule — hysteresis band (prevents demand oscillation):
#   Enter:  market_price > 1.5 × default_price  → add market to global list
#   Exit:   market_price < 1.2 × default_price  → remove market from global list
#   Dead zone [1.2×, 1.5×]: no state change (list membership persists)
#
# Demand-side conservation is handled in SOL_goods_demand_values.txt (script_values):
#   - Scarce good keeps 40% baseline demand (multiply 0.4).
#   - Missing baseline spending is redistributed to group alternatives.
#   - Redistribution coefficients use default_market_price to keep
#     group baseline total expenditure constant under any scarcity mix.
#
# Storage: 54 global variable lists (SOL_<good>_scarce_markets), one per good.
# Each list contains the market scope objects where that good is currently scarce.
# Pop demand checks: location.market ?= { is_target_in_global_variable_list = { ... } }

                limit = {
                    NOT = {
                        is_target_in_global_variable_list = {

# Computes per-location scarcity flags for all pop-demand substitute groups.
#
# Unified rule — hysteresis band (prevents demand oscillation):
#   Enter:  market_price > 1.5 × default_price  → flag ON
#   Exit:   market_price < 1.2 × default_price  → flag OFF
#   Dead zone [1.2×, 1.5×]: no state change (flags persist from last evaluation)
#
# Demand-side conservation is handled in z_SOL_pop_demand.txt:
#   - Scarce good keeps 40% baseline demand (multiply 0.4).
#   - Missing baseline spending is redistributed to group alternatives.
#   - Redistribution coefficients use default_market_price to keep
#     group baseline total expenditure constant under any scarcity mix.

# Source Files

This directory contains the mod's source code and game scripts.

## Structure

- `common/` - Game data definitions (countries, cultures, religions, etc.)
- `events/` - Event scripts
- `gui/` - User interface files
- `localization/` - Translation files (.yml format)

## Guidelines

- Follow EU5 scripting conventions
- Use UTF-8-BOM encoding for all text files
- Test all changes in debug mode before committing
- Document complex logic with comments
